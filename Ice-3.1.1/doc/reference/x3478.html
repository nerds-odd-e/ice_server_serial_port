Ice::NoObjectFactoryException

Overview

local exception NoObjectFactoryException
    extends 
	MarshalException

This exception is a specialization of MarshalException that is raised if no suitable object factory was found during object unmarshaling.

See Also

ObjectFactory, Communicator::addObjectFactory, Communicator::findObjectFactory
